# project-management-tool
An open-source trello/JIRA alternative for streamlining the product development and staging the process, built on MERN stack.

## Tech Stack
- ReactJS (FrontEnd)
- MongoDB (Database)
- NodeJS (RTE)
- ExpressJS (Backend)
- Tailwind (for styling)
- Deployment (AWS)
- Docker (for containerization)
- Postman (for testing)
